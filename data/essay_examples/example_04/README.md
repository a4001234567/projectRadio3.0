# Example 4 — Moore Matrix, Weight-1 (Truncated vs. Untruncated vs. PEG)

Moore construction with weight-1 circulants. Compares truncated, untruncated,
and PEG variants. The truncated form requires (I - H) to be block MDS for the
theoretical guarantee; the untruncated case satisfies this exactly, while the
truncated case is expected to be approximately equivalent in practice.
